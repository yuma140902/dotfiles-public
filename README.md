# dotfiles

## インストール方法

```
usage: dotfiles [-h] [-y] [-v] {install,list,info} ...

options:
  -h, --help           show this help message and exit
  -y, --noconfirm      ユーザーに入力を求めない
  -v, --version        show program's version number and exit

Sub Commands:
  {install,list,info}
    install            モジュールをインストールする
    list               モジュールを一覧表示する
    info               モジュールの情報を表示する
```
